using the html provided below, write a dice rolling program that does the following:

  1. Generates a random number between 1 and 24 when #roll_dice_button is clicked
  2. Compares the generated random number with the value of #my_bet
    • if the value of #my_bet is blank fire an alert saying "value of input cannot be blank". The program should NOT run any further.
    • If the value of #my_bet is less than the the number generated randomly, alert "sorry you guessed too high"
    • If the value of #my_bet is greater than the number generated randomly, alert "sorry you guessed to little"
